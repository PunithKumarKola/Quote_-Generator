# Quotable — Random Quote Generator

A minimal React + TypeScript + Tailwind CSS app built on TanStack Start.

## Run locally in VS Code

Requirements: [Node.js 20+](https://nodejs.org) and [Bun](https://bun.sh) (npm also works).

```bash
# 1. Install dependencies
bun install        # or: npm install

# 2. Start the dev server
bun dev            # or: npm run dev

# 3. Open the printed URL (usually http://localhost:8080)
```

Open the folder in VS Code with `code .` and use the built-in terminal.
Recommended extensions: **ESLint**, **Prettier**, **Tailwind CSS IntelliSense**.

## Scripts

- `bun dev` — start dev server
- `bun run build` — production build
- `bun run preview` — preview production build
